
# Portfolio Site — Single Page

This repository contains a single-page static portfolio with the following sections: Home, Services (with slideshow), Projects, and Contact.

Preview locally — run a simple HTTP server from the `portfolio-site` folder:

```bash
# From the folder e:/xhama/Ict projects/portfolio-site
python -m http.server 8000
# Open http://localhost:8000 in your browser
```

Notes:
- The site uses a black / red / white color scheme.
- Services slideshow uses placeholder images from picsum.photos; replace with your ICT images in `images/` or update URLs.
- Projects section contains placeholders where you can add real images and links to GitHub and Google Drive.
